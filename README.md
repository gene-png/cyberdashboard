# Zero Trust Maturity Assessment Dashboard

A Flask-based dashboard for assessing organizational maturity against Zero Trust principles.

## Status
Under active development. See `docs/STATE.md` for current status and `CHANGELOG.md` for version history.

## Development

This project is built and maintained inside a Docker dev container using [Claude Code](https://docs.claude.com/en/docs/claude-code/overview).

### Quick start

1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop) and [VS Code](https://code.visualstudio.com).
2. Install the **Dev Containers** extension in VS Code.
3. Clone this repo and open it in VS Code.
4. When prompted, click **"Reopen in Container"**.
5. In the container terminal, run `claude` to start a Claude Code session.

### Project conventions

Project context, rules, and workflow are defined in [`CLAUDE.md`](./CLAUDE.md). Architectural decisions live in [`docs/decisions/`](./docs/decisions/).
